# README - Admin Frontend

## Muc tieu UX

Giao dien admin uu tien giam sat he thong va thao tac nhanh khi co su co, khong phuc vu quy trinh tao content hang ngay.

## Cac man hinh admin can co

1. `Admin Dashboard`
   - Tong user active.
   - Tong campaign dang chay.
   - AI token usage theo ngay.
   - So workflow job failed.
2. `User Management`
   - Danh sach user, trang thai.
   - Khoa/mo tai khoan.
3. `AI Usage Monitor`
   - Bang usage theo model/provider.
   - Canh bao campaign bat thuong.
4. `Workflow Operations`
   - Danh sach job loi.
   - Retry thu cong.
5. `Audit Logs`
   - Timeline hanh dong admin.

## Dieu huong

- Sidebar admin rieng (duong dan `/admin/*`).
- User thong thuong khong thay menu admin.

## Nguyen tac giao dien

- Giong style AIMAP hien tai: kho, ro, uu tien du lieu.
- Tap trung table, filter, bulk actions.
- Hien thi ro thong bao canh bao va ket qua thao tac.
